# Dollar Identify

Static Teachable Machine image classifier for identifying U.S. dollar bill denominations.

Classes:

- $1
- $2
- $5
- $10
- $20
- $100

The app runs in the browser with TensorFlow.js and loads the local model files from `assets/dollar-model/`.

## GitHub Pages

Publish this folder as a GitHub Pages site from the `main` branch and root directory. The app entry point is `index.html`.

## Representative Images

The denomination visuals in `assets/dollar-denominations/` are non-realistic educational cards, not printable currency replicas.
