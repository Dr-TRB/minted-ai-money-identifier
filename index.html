    Privacy

    Photos are processed on this device in the browser. This app does not intentionally upload or store images; it only downloads the model and library files needed to run the classifier.

    Class details

    Edit CLASS_DETAILS in dollar-identify.js. Match each key to a Teachable Machine class label, then add a representative image URL, description, and feature list.

    Privacy

    Images from the camera or photo picker are analyzed locally in the browser with TensorFlow.js. The app does not send photos to a server or save them. The browser does request the Teachable Machine model and JavaScript libraries from external providers so the classifier can run.
